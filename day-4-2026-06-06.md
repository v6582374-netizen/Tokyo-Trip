### 矢口书店（矢口書店 / Yaguchi Book Store）

- **类型**：书店 / 老铺（古本 / 旧书专门店；戏剧 / 电影 / 日本文学专精）
- **位置**：东京都千代田区神田神保町 2-5-1；最近车站 东京 Metro 半蔵门线 / 都营三田线 / 都营新宿线「神保町」站 **A6 / A7 出口步行 1 分钟**；JR 中央 / 总武线「水道桥」站步行 10 分钟、东京 Metro 千代田线「新御茶水」站步行 8 分钟。店面就在靖国通南侧支路、白山通西侧的街角处，**外推人行道的连绵木造书架** 是远远就能识别的标志。
- **历史与背景**：矢口书店创业于 **1918 年（大正 7 年）**，由初代矢口长太郎在神保町开设；恰逢神保町作为「日本人文出版黄金时代」起步的关键节点——**明治末期至大正年间**（1900–1925），神保町因周边明治大学、中央大学、日本大学、专修大学、东京大学（本乡）等高校群密集而催生出"学生城 + 古书街"的双重格局，矢口书店即在此潮流中开店。**1923 年（大正 12 年）9 月 1 日关东大震灾**中，神保町几乎全街烧毁、矢口书店初代店面亦付之一炬；翌年震灾重建潮中，矢口书店在原址重新开张，成为神保町战前老铺中少数延续创业地址的字号。**第二次世界大战末期 1945 年东京大空袭**神保町核心区奇迹般避开焚毁（一说与街区内大量"古书纸箱"被列为"防火资材"优先保护有关），战后矢口书店得以原貌延续；**昭和 20–40 年代** 战后古本街复兴期，二代目矢口家进一步把店铺主营聚焦到「**戏剧（演剧）/ 电影（映画）/ 日本古典文学**」三大专门领域，与隔壁后来成立的「**东京古书会馆**」（1967 年开馆，全日本古书业的总部 + 古书拍卖会场）并列成为神保町最具代表性的两块"百年品牌招牌"。今天矢口书店由**三代目矢口家继承**经营，店头那一连串外推到人行道上、长约 8 米的露天木造书架（"店頭ワゴン棚"）已成为**神保町最易识别的视觉地标之一**——许多 NHK / TBS / 旅行频道介绍神保町古书街的 B-roll 镜头几乎都包含矢口书店的木架特写。店内深进约 12 米的窄长空间里，**戏剧 / 电影专门区** 收藏了大量绝版的日本舞台剧本、能 / 狂言 / 歌舞伎台本、战前 / 战后映画杂志 (例如「キネマ旬報」全卷)、电影海报与剧场节目册；**日本古典文学区** 收藏从万叶集、源氏物语、近世戏作（井原西鹤 / 曲亭马琴）到近代文学（漱石 / 鸥外 / 谷崎 / 三岛）的初版本、限定本与古文书。一句话：矢口书店是"神保町古书街的脸面"，CSV 备注里被特别点出的「标志性的街边书架」即指此外立面。
- **看点**：
  1. **街边木造书架（店頭ワゴン棚）**：店面外推人行道上、长约 8 米的连绵露天木架，约 1,000–1,500 册百元级古本（多数 ¥100–500 / 本）按主题摆放，是矢口书店最易识别的外立面、也是神保町古书街的视觉标志；从对街拍木架与店招合影是经典构图。
  2. **戏剧 / 电影专门区**：店内中段约 5 米长的专架，收藏战前 / 战后日本舞台剧本、能 / 狂言 / 歌舞伎台本、映画杂志全套（如「キネマ旬報」从 1919 年创刊以来的部分卷次）、剧场节目册、电影海报缩印本；价位从 ¥800（普通杂志旧号）到 ¥30,000+（绝版剧本初版）。
  3. **日本古典 / 近代文学区**：店内深处的窄长书墙，收藏漱石 / 鸥外 / 谷崎 / 三岛 / 川端的初版本、限定本、签名本，以及万叶集 / 源氏物语 / 近世戏作的旧版与古文书；初版漱石《吾辈は猫である》全 3 册曾在此流转，价位 ¥10,000–200,000+。
  4. **古本订单流程（注文 / 取置）**：架上没有的特定书目可向店主请「注文」（订单查询）——三代目矢口先生与神保町同业网络可在 1–7 日内调来；店头黑板会写出最近到货的"注目本"清单，是神保町古书业"店主选品 + 同业流转"的活样本。
  5. **店招与玻璃窗**：店招"矢口書店"四字为传统招牌字体（**待用户实地确认是否为初代手写**），与玻璃窗内堆叠到天花板的旧书墙构成"昭和时代古本店"原貌——是日剧 / 电影里"神保町场景"的常用取景地之一。
- **推荐玩法**：**10:30 开门后立即抵达**（神保町古书店 11:00 后客流上来、矢口书店店内仅约 12 米深的窄长空间易拥挤）→ 先在街对面 / 人行道边拍 1 张木造书架 + 店招的全景外观；→ 翻 30–45 分钟（先翻店头百元木架 → 再进店看戏剧 / 电影专门区 → 末尾扫一眼古典文学墙）；→ **选 1–2 本旧书作纪念**（百元木架的旧映画杂志 ¥100–500 / 本是性价比最高的"神保町手信"，比新书店买图书更有"古书街气息"）；→ 出店后向北沿白山通走 3 分钟到三省堂 / 东京堂书店做 5–10 分钟顺路打卡，再转入咖喱街午饭。**全段在矢口书店内停留 30–45 分钟即可**，不必硬把整条神保町古书街逛完（200 家书店 1 天也逛不完）。

### 新宿御苑（新宿御苑 / Shinjuku Gyoen National Garden）

- **类型**：庭园 / 公园（国营 / 国民公园；日 + 英 + 法三式庭园复合体）
- **位置**：东京都新宿区内藤町 11；最近车站 **东京 Metro 丸之内线「新宿御苑前」站 1 / 2 出口步行 5 分钟**（约 350 m）至新宿门 / 大木户门；亦可东京 Metro 丸之内线 / 副都心线 / 都营新宿线「新宿三丁目」站 E5 出口步行 5 分钟（约 400 m）至新宿门；JR / 私铁 / 地铁 各线「新宿」站南口步行 10 分钟（约 800 m）至新宿门；都营新宿线「新宿御苑前」（与丸之内线同站名不同位置 —— 待用户实地确认换乘出口）。园区南北约 1.1 公里、东西约 700 米、**总面积 58.3 公顷（583,000 m²）**，是新宿最大的连续绿地。
- **历史与背景**：新宿御苑的土地起源于 **江户时代初期 1590 年代**——德川家康入府关东后将此地（旧名「内藤町」）赐给徳川家家臣 **内藤清成**（信州高远藩内藤家初代）作为下屋敷领地，因此整片土地最初是**信州高远藩内藤家的江户下屋敷**（约 22 万坪 / 72 公顷），是江户外缘最大的大名屋敷之一；**1698 年（元禄 11 年）** 内藤家把屋敷北缘的甲州街道宿场让出供幕府新设宿驿，即「**内藤新宿**」（今"新宿"地名的由来——"新设的宿场"）。**明治维新后 1872 年（明治 5 年）** 内藤家把屋敷上缴政府，**1879 年（明治 12 年）** 改为「**内藤新宿试验场**」（农林水产部门管辖、引种欧美果树 / 蔬菜的农业试验场）；**1906 年（明治 39 年）5 月 26 日** 经天皇敕令改建为「**新宿御苑**」对皇室开放——以法国凡尔赛宫为参考的"皇室庭园"——由法国造园家 **Henri Martinet（亨利・马丁内）** 主导设计西式整型庭园（西芝生）+ 英式风景庭园（中央芝生）+ 日本庭园（旧御凉亭一带）三式合一布局，**这是日本第一座"日 + 西"复合式皇家庭园**。**1949 年（昭和 24 年）5 月 21 日** 战后由 GHQ 命令开放为「**国民公园**」对公众开放，由 **环境省**（前身厚生省）管辖至今；**1958 年（昭和 33 年）** 园内大温室（旧温室）完工、**2012 年** 现代温室（新温室）开放，是现今日本最具规模的兰花 / 热带花卉温室之一。园内**樱花约 1,100 株 / 65 个品种**（日本最丰富的樱花品种园之一），从 2 月「寒樱」到 4 月下旬「关山 / 八重樱」连续开放近 3 个月，是东京都心赏樱代表地（与上野公园 / 千鸟之渊并列）；6 月梅雨季虽不在樱花期，但温室玫瑰 / 兰花 / 蜘蛛兰、母与子之森 / 玉藻池畔的紫阳花仍构成可观花景。一句话：新宿御苑是**江户大名下屋敷 → 明治皇家庭园 → 昭和国民公园**这一三层叠加的活样本，本质上不是"普通公园"，而是日本近代庭园史的代表作。
- **看点**：
  1. **法国式整型庭园（Jardin à la française / 西芝生）**：园西侧约 6 公顷的对称式法式整型庭园 —— **巴黎凡尔赛 + 杜伊勒里宫**为参考，由 Henri Martinet 1906 年设计的"日本第一座法式整型庭园"，中央喷水池 + 两侧法国梧桐林荫道 + 修剪整齐的草坪几何块；秋季银杏 / 法国梧桐变色时是关东最美的落叶街道之一。
  2. **英国式风景庭园（English landscape garden / 中央芝生）**：园中央约 12 公顷的开阔草坪 + 不规则水池 + 自然式树丛 —— 英国 18 世纪 Capability Brown 风格，是东京少数允许"草地野餐 + 摊毯子睡午觉"的大型公共绿地（**禁止酒精 / 烟火 / 球类**），周末家庭群聚的画面是新宿地表的"和平象征"。
  3. **日本庭园（旧御凉亭 + 玉藻池）**：园东侧约 8 公顷的池泉回游式日本庭园 —— 中央有 1928 年（昭和 3 年）昭和天皇结婚祝贺时由台湾总督府寄赠的「**旧御凉亭**」（中国闽南风格的木造凉亭，亦称"台湾阁"），是园内唯一登录有形文化财；周边玉藻池畔 6 月有紫阳花、9 月有彼岸花。
  4. **大温室（新温室）**：园南侧的现代玻璃温室（2012 年开放）—— 收藏约 2,700 种热带 / 亚热带植物（兰花 / 食虫植物 / 椰子 / 香蕉 / 日本绝灭危惧种），常年 25–30°C，6 月梅雨季是最舒服的避雨场所。
  5. **樱花约 1,100 株 / 65 品种**：从 2 月「**寒樱 / 河津樱**」、3 月下旬「**染井吉野**」、到 4 月「**关山 / 一叶 / 八重樱**」连续 3 个月——本次行程 6 月不在樱花期，但**樱花树本身**（树形 + 树皮 + 嫁接标牌）仍可作为"日本樱花博物馆"性质的植物学打卡。
  6. **母与子之森**：园北端的杂木林散步道 —— 1968 年昭和天皇 / 香淳皇后为纪念皇太子结婚而命名，林中保留江户时代内藤家屋敷遗树（武藏野落叶林原貌），是园内最少游客、最幽深的一段。

### 歌舞伎町（歌舞伎町 / Kabukicho）

- **类型**：商业街 / 夜游（亚洲最大规模的红灯 / 娱乐街区；现代国际化亚文化夜场代表）
- **位置**：东京都新宿区歌舞伎町 1–2 丁目；最近车站 **JR / 私铁 / 地铁各线「新宿」站东口步行 5 分钟**（约 400 m）至歌舞伎町一番街南口；亦可 西武新宿线「西武新宿」站正面口出闸即入歌舞伎町北端步行 2 分钟、东京 Metro 副都心线 / 都营大江户线 / 丸之内线「新宿三丁目」站 B11 / B13 出口步行 5 分钟、都营大江户线「东新宿」站 A1 出口步行 5 分钟。街区南北约 600 米、东西约 400 米，**约 0.24 km² 的紧凑街区集中了约 3,000 家餐饮 / 风俗 / 娱乐设施**（日本最高密度），是新宿东口的核心商业街区。
- **历史与背景**：歌舞伎町的土地起源于 **江户时代** 内藤新宿宿场的外围低洼地带（旧名「角筈」），是周围村落的稻田 + 沼泽地带；**明治末期至昭和初期** 此地为新宿西北边的"水稻地区 + 杂居住宅区"，没有商业街性格。**1945 年（昭和 20 年）东京大空袭** 中此地被彻底烧毁，战后 GHQ 占领期由当地町会会长 **铃木喜兵卫** 主导战后复兴计划——**1948 年（昭和 23 年）4 月** 町会决定在此**新建一座大型歌舞伎剧场**作为街区核心，并据此把街区命名为「**歌舞伎町**」（"歌舞伎座"街），但**剧场建设最终因资金 + 用地问题终未实现**——歌舞伎町因此成为**日本唯一一个"以从未建成的剧场命名"的街区**。**1950 年代** 街区因新宿站东口的人流红利迅速发展为新宿首屈一指的商业街，但**红线指定区**（売春防止法施行前的合法红灯区）+ 战后黑市 + **韩国华侨街**（战后从中国大陆 / 朝鲜半岛流亡来日的华侨 / 朝鲜族在歌舞伎町集中开设饮食店 + 玩具批发 + 黑市贸易，今天的"东京韩国城"新大久保即从歌舞伎町西北边外溢形成）+ **黑社会**（山口组 / 住吉会 / 稻川会三大组的关东支部均在歌舞伎町设事务所）+ 风俗营业等多重力量交织，在 **1960–80 年代** 形成"日本最复杂的城市生态"，被警察厅长期列为"全国治安重点"。**1990 年代起** 街区开始国际化转型——**亚洲（中 / 韩 / 越 / 泰）+ 欧美**留学生与劳动者大量涌入开店，传统风俗业转向**主持俱乐部（ホストクラブ）/ 牛郎店 / 二次元主题餐饮（女仆咖啡 / 执事咖啡）/ 电竞酒吧 / Karaoke**等更现代的形态。**2002 年「新宿 KOMA 剧场」拆除** 后街区进入大规模再开发期，**2015 年「新宿东宝大厦」（Hotel Gracery 新宿 + TOHO 影院 + 屋顶哥斯拉雕像）** 开业、**2023 年 4 月「东急歌舞伎町塔」（Tokyu Kabukicho Tower）** 开业（隈研吾设计的 48 层 225 米地标摩天楼，含 NAMCO LAND / 109 Cinemas Premium / Bellustar Tokyo / Hotel Groove Shinjuku 等设施），是 21 世纪歌舞伎町从"红灯街"向"国际化娱乐综合街区"转型的代表项目。今天歌舞伎町俗称「**眠らない街 / 不眠之街**」（"Neverland"或"Sleepless Town"），24 小时人流不断，是东京夜文化的视觉地标——大量被韩剧 / 中国大陆电影（《唐人街探案 3》/《东京物语》/《新宿事件》）取景，亦是国际游客打卡新宿夜景的核心点。CSV 备注「歌舞伎町」即指此街区整体（用户没有特定子点）。
- **看点**：

  6. **二次元主题街（女仆咖啡 / 执事咖啡 / 电竞酒吧）**：街区东侧靠近 Golden-Gai 的小巷里集中了 20+ 家女仆咖啡 / 执事咖啡 / 二次元主题餐饮 / VR 游戏中心，是歌舞伎町 21 世纪国际化转型的代表 —— 与秋叶原同源但更"夜场化"。

### 新宿黄金街（新宿ゴールデン街 / Shinjuku Golden-Gai）

- **类型**：商业街 / 夜游（约 280 间木造小酒场密集的"昭和迷宫"；「深夜食堂」取景地）
- **位置**：东京都新宿区歌舞伎町 1 丁目 1–6 番；最近车站 **东京 Metro 副都心线「东新宿」站 A1 出口步行 7 分钟**（约 500 m）；**东京 Metro 丸之内线 / 副都心线 / 都营新宿线「新宿三丁目」站 E1 出口步行 5 分钟**（约 400 m）；**JR / 私铁 / 地铁各线「新宿」站东口步行 10 分钟**（约 800 m）；**西武新宿线「西武新宿」站正面口步行 5 分钟**（约 400 m）；**都营大江户线 / 副都心线「东新宿」站 A1 出口步行 7 分钟**。街区**南北约 100 米、东西约 70 米**（约 0.007 km² / 7,000 m²，仅相当于一个标准足球场大小），但内部以 **6 条窄巷 + 200+ 栋 2–3 层木造长屋 + 约 280 间小酒场（10 平米以下小店为主）** 形成超高密度的"昭和迷宫"，是世界上**单位面积酒场密度最高的街区**之一；位置上**夹在歌舞伎町（西）+ 花园神社（东）+ 区役所通（北）+ 靖国通（南）四线之间**，从歌舞伎町一番街东侧步行 5 分钟即可进入。
- **历史与背景**：新宿黄金街的土地起源于 **江户末期至明治期**，原为新宿外围低洼湿地与花园神社境内地的边缘地带，没有商业街性格；**1945 年（昭和 20 年）东京大空袭** 中此地与歌舞伎町同被烧毁。**战后 1946–1949 年** GHQ 占领期，街区作为"**青空市场 / 露天黑市**"（**新宿マーケット**的一部分）由战后归国军人 + 难民 + 家庭主妇集中露天摆摊售卖食物 / 杂货 / 黑市货，是新宿东口与西口大型黑市群的最东端部分。**1950 年（昭和 25 年）** 黑市规范化后此地与歌舞伎町区分独立，被指定为「**红线（赤線）地区**」（売春防止法施行前的合法红灯区）—— 街区从黑市转为约 80 间「特殊饮食店」（红线 = 政府公认色情饮食店）+ 100+ 间「青线」（政府未公认但默许的色情店）的混合街区，**这是黄金街作为夜场街的起点**。**1958 年（昭和 33 年）4 月 1 日 売春防止法（红线法）全面实施** 后所有红线 / 青线饮食店被禁止从事色情营业，全街约 200 间店铺**集体转型为只提供酒水 + 简餐 + 谈话的"小酒场（飲み屋 / バー）"**——这一"红线 → 小酒场"的转型奠定了今日黄金街的格局：每间店面仅 8–15 平米、容纳 6–10 人、店主（多为女性 / "妈妈桑"）+ 常客 + 偶尔的访客构成的"私人客厅式酒吧"。**1960–80 年代** 街区因房租低 + 店主性格强 + 客层"硬核"而吸引大量**漫画家 / 作家 / 编剧 / 演员 / 摄影师 / 学生运动家**聚集——三岛由纪夫 / 寺山修司 / 唐十郎 / 桑原甲子雄 / 中上健次 / 矢作俊彦 / 矢野龙司 / 东海林さだお 等昭和文化人都是这里的常客，黄金街从而获得「**東京最後のスラム**」（东京最后的贫民窟）+「**文化人の集う街**」（文化人聚集街）双重昵称。**1990 年代日本泡沫破裂期** 大型不动产开发商（西武 / 三井等）多次试图整体收购拆除黄金街改建摩天楼，但因**复杂的地权 + 店主联合抵抗**未能实现；**2004 年起街区被新宿区指定为"昭和景观保全区"**，地权与建筑外观法律上受保护。**2010 年代起** 街区因旅游业国际化转型——**安倍晋三政府"观光立国"政策** + **NHK 深夜剧《深夜食堂》（2009 起）取景地**双重加持下，国际游客（韩 / 欧美 / 中 / 港台）大量涌入，**约 30%–40% 的店家开始接受国际客人**（部分挂出"Foreigners Welcome / 観光客 OK"招牌），但**多数老店家仍坚持"日本人 only / 熟客 only"的硬核作风**——不熟客可能在门口被店主礼貌拒入（"今日は満席です / 本日已满席"，即不收新客）。今日黄金街是**昭和战后小酒场街唯一完整保留**的街区，也是**日本夜文化的活化石**——CSV 备注「深夜食堂的饭屋就设定在这个地方，这次一定要去」即指此街区，《深夜食堂》漫画 / 影视的取景地原型即黄金街内的多家小酒场（如「**しんぱち**」「**鍋ヤ**」等真实店铺）。
- **看点**：
  1. **6 条平行窄巷的木造长屋格局**：街区内"花园 1 番街 / 2 番街 / 3 番街 / G2 通り / G1 通り / 仲通り" 6 条窄巷（每条宽 1.2–2 米，仅容 1–2 人并行）+ 两侧 2–3 层木造长屋 —— 这是**昭和 30 年代日本城市原貌的活样本**，在 2025 年的东京已是绝无仅有；夜晚红灯笼 + 木门 + 招牌的密集排列是黄金街最易识别的视觉地标。
  2. **约 280 间小酒场密集排列**：每间店面仅 8–15 平米、容纳 6–10 人，**单位面积酒场密度世界第一**；店主多为 50–80 岁的"妈妈桑"，店内装饰个性强烈（爵士 / 摇滚 / 文学 / 电影 / 漫画 / 同性恋亚文化等不同主题）。
  3. **「深夜食堂」取景地**：NHK 深夜剧《深夜食堂》（小林薫主演 / 2009–2019）+ Netflix《深夜食堂：Tokyo Stories》（2016–2019）的"街道全景镜头"+ 部分内景在黄金街拍摄；街内"しんぱち"（一番街内）+「鍋ヤ」（仲通り）等店铺为剧组顾问。
  4. **花园神社（Hanazono Shrine）**：街区东侧 30 米的江户期勧请神社（本殿 24 小时开放）—— 新宿总镇守、商売繁盛 / 艺能上达保佑神，5 月例大祭与 11 月酉之市为新宿最大祭典；本日 6/6 不在祭典期，但本殿境内可顺路参拝。
  5. **"お通し代 / 席料 / Charge"文化**：黄金街多数店家**先收 ¥500–1,500 的お通し代 / 席料**（不点酒水也收，作为"占座 + 小菜服务费"），是初次访客最易被吓到的"非透明定价"；进店前**必看门口立牌的"Charge ¥XXX"标示**，无标示或仅日文标示的店家国际客需谨慎。
  6. **门口"Foreigners Welcome / 観光客 OK"招牌**：约 30%–40% 的店家挂明确招牌欢迎国际客（菜单有英文 / 价目表透明），是国际游客首选；多数老店家**无此招牌、坚持日本人 only / 熟客 only**，进门前店主可能礼貌拒入。
- **推荐玩法**：
  1. **22:00 后入街最佳**（多数店家 18:00–翌日 05:00 营业，22:00 后才进入"文化人 + 妈妈桑闲聊"的硬核氛围）；
  2. **从 G2 通り南端（靠近靖国通）入街** → 沿 G2 通り北行穿过 6 条窄巷 → 在 仲通り东端出（靠近花园神社）→ **夜游本身（不入店）30–60 分钟即可**——拍木造长屋 + 红灯笼 + 招牌密集排列的夜景；
  3. **如想进店体验** → **优先选门口挂"Foreigners Welcome / 観光客 OK / English Menu"招牌的店家**（如「**ニュー鈴屋**」「**Albatross G**」「**鼎 KANAE**」「**Cremaster**」等都是国际客口碑店，菜单 + Charge 透明）→ 入店点 1 杯啤酒（¥800–1,200）+ お通し代（¥500–1,000）→ 与店主 / 邻座聊 30–60 分钟；
  4. **如不入店** → 仅街内夜游 + 在花园神社境内静坐 5 分钟 → 经歌舞伎町返新宿站。
- **票务 / 营业时间**：街区**免费 + 24 小时开放**（公共道路）；**多数店家 18:00–翌日 05:00 营业**（部分店家 22:00 后才开门、04:00 闭店——是新宿最深夜的酒场街）；周日 + 国定假日多数店家定休（**6/6 周六多数店家正常营业**）；个体店家 Charge ¥500–1,500 + 酒水 ¥800–1,500 / 杯。
- **交通到达**：
  - **从前一地点 歌舞伎町 / Bankara 拉面新宿歌舞伎町店**：从一番街拱门东行 200 米 → 穿过 区役所通 / 新宿东口职业安定所 → 步行 5 分钟即入黄金街西端；亦可从花园神社西端入街。

  1. **"お通し代 / 席料 / Charge ¥500–1,500" 必看门口立牌**：黄金街多数店家**先收 ¥500–1,500 的 Charge**（不点酒水也收），这是日本酒场合法的"占座费 + 小菜服务费"；进店前**必看门口立牌的"Charge ¥XXX"或"席料 ¥XXX"标示**，**无标示或仅日文标示的店家国际客谨慎入**；如无英文菜单 + Charge 不透明，**婉拒退出**即可（"あ、また来ます / 我下次再来"）。

- **历史与背景**：Mandarake（まんだらけ）的创业故事是日本御宅文化史最重要的章节之一——**1980 年（昭和 55 年）2 月**，**漫画家古川益藏**（古川益蔵 / Furukawa Masuzo，1948–）在**中野百老汇 4 楼**开设了一家仅 8 平米的中古漫画店，命名为「**まんだらけ**」（Mandarake，源自佛教用语「曼陀羅華」+ 「だらけ」"满是…"的合成词，意为"漫画满满"），是日本**第一家专营中古漫画 + 同人志 + ACG 周边的连锁前身**。古川益藏出身手塚治虫主宰的「虫プロ商事」，在 70 年代末日本漫画产业从"杂志连载 → 单行本市场"转型期发现了"**中古漫画 + 绝版同人志 + 70 年代昭和动漫周边**"被市场严重低估的商机，遂在中野百老汇 4F（当时百老汇的空置店铺极多、租金极低）以"**收购旧家庭书柜中的漫画 + 转售给收藏家**"的循环模式起步；**1987 年**首次在中野百老汇内扩张为 3 间分店（漫画店 + 玩具店 + 同人志店），形成**"主题分店化经营"**的雏形——这一模式后来成为 Mandarake 全国扩张的核心战略，与同时期秋叶原的虎之穴 / Animate 等主流"综合书店式"竞争对手形成根本差异。**1994 年品牌全国扩张** 至涩谷、池袋、大阪、福冈等地；**2000 年代起在中野百老汇内进一步细分**为 30+ 主题分店——其中包括 CSV 备注里特别提到的 **「**门**（モン / Mon）」店**（古董玩具专门店，专卖战前 / 战后昭和 30–50 年代的铁皮玩具 / 软胶玩具 / 万代 / 田宫 / Takara 等老厂的初期产品 / 限定版 / 罕见品）、**「**大怪兽**（大怪獣 / Daikaiju）」店**（怪兽软胶专门店，专卖东宝哥斯拉 / 圆谷奥特曼 / 大映卡美拉 / 东映幪面超人等战后特摄系列的怪兽形象软胶玩偶）、**「**Galaxy**（ギャラクシー）」店**（怀旧游戏机原装卡带专门店，专卖 FC / SFC / N64 / GameBoy / MegaDrive / NeoGeo 等 1980–2000 年代家用机的原装卡带 / 主机 / 周边）；这三家是 CSV 备注 ⼿点的代表性分店，但并非全部——其他还有「**特撮**」「**LIVE**（同人志 / 限定本）」「**WARTOI**」「**カードショップ**」「**SF**」（科幻单行本）「**少女漫画**」「**Cosplay**」「**コスチュームコーナー**」「**着物**」「**鉄道模型**」等共 30+ 间散布在 2F–4F 各层的主题店。**与秋叶原 Complex 的选品分化**：Mandarake Complex 秋叶原店（2003 年开业、Day 3 已访问）以"**通俗ACG动画 / 当代同人志 / 现役漫画 / Cosplay / 偶像周边**"为主，面向 10–25 岁的当代御宅；**中野店本店**则以"**昭和怀旧 / 古董玩具 / 怪兽软胶 / 怀旧游戏机原装卡带 / 绝版漫画初版本 / 战前少女漫画**"为主，面向 30–60 岁的"**核心收藏家 / 资深御宅**"——这种"秋叶原 = 当代 / 中野 = 怀旧"的二元分化是 Mandarake 在日本御宅市场上的战略定位。**国际游客占比**：根据中野百老汇商城 2023 年公开数据，Mandarake 系分店的**国际游客占比约 40%**（远高于 Complex 秋叶原店的 30% 与涩谷店的 25%）—— 这一比例反映了"**中野百老汇 = 全球昭和御宅文化的 Mecca**"的国际地位，BBC / NHK World / 法国 Arte 频道等多次将中野百老汇与秋叶原电气街并列为"东京御宅双圣地"。今日 Mandarake 中野店是**Mandarake 系全国 30+ 分店的发源地与精神图腾**——CSV 备注「中野百老汇是 Mandarake 的发源地和精神图腾」即指此。
