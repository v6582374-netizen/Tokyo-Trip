# Euro Live（涩谷剧场）（ユーロライブ / Euro Live）


- **类型**：演出 / 活动会场（Euro Space 旁的中型实验剧场，约 200 席；4 个补丁外观点之一，按地理纳入 Day 6 上午段）；**展期已结束（6/2–4），本次行程仅外观打卡**

- **历史与背景**：Euro Live（ユーロライブ）于 2006 年开馆，是涩谷円山町 KINOHAUS 大楼 4F 的中型实验剧场，由日本独立电影 / 实验艺术发行公司 Euro Space（株式会社ユーロスペース，1982 年开业的同名艺术影院在同栋大楼 3F + 4F）运营。KINOHAUS 大楼是 2006 年由 Euro Space 创办人堀越谦三主导改建的"涩谷独立电影 / 实验艺术综合楼"——3F + 4F 是 Euro Space 艺术影院（约 145 + 79 席），4F 同层另设 Euro Live 实验剧场（约 200 席，舞台 / 影像 / 实验音乐 / 现场表演的可变换空间）。Euro Live 自开业以来承办大量独立电影首映 / 短片节 / 实验音乐 / 戏剧 / 现代舞 / 视觉艺术演出，是涩谷円山町亚文化区的"地下文化中心"之一，与道玄坂北面的 Bunkamura（东急集团运营的复合文化中心）+ 公园通顶端的 LINE CUBE SHIBUYA（涩谷区立公会堂）形成涩谷"主流—独立—公共"三段演出场分工。本次行程的 6/8 涉及 SSFF & ASIA 2025 短片国际电影节—**Euro Live 承办的是 6/2（周一）至 6/4（周三）的「亚洲国际竞赛 / 日本国内竞赛」单元的连续放映 + 映后交流，展期已于 6/4 结束**。本日 6/8 已过末日 4 天，**仅外观打卡**——按 §6.6 规则文档明示「展期 6/2–4 已结束，本日仅外观打卡」。
- **看点**：
