# 旧岩崎邸庭园（旧岩崎邸庭園 / Kyū-Iwasaki-tei Gardens）

- **类型**：庭园 / 历史建筑（明治洋馆 + 和馆 + 撞球室三栋一体）
- **位置**：东京都台东区池之端 1-3-45（不忍池西岸南端，距上野公园不忍口南行约 8 分钟）；最近车站 东京 Metro 千代田线「汤岛」站 1 番出口步行 3 分钟、东京 Metro 银座线 / 都营大江户线「上野广小路 / 上野御徒町 / 上野」站步行 10 分钟、JR「御徒町」站步行 15 分钟。

## 历史与背景

旧岩崎邸庭园是三菱财阀创始人岩崎弥太郎家族（弥太郎之子 三菱第三代社长 **岩崎久弥（1865–1955）**）的本邸旧址，所在用地原是江户时代越后高田藩榊原家的中屋敷，1878 年（明治 11 年）由岩崎弥太郎购入并作为家族本邸。**岩崎久弥是岩崎弥太郎的长子，也是三菱财阀的第三代掌门人**。1896 年（明治 29 年）由当时受邀来日的英国建筑师 **约西亚·康德尔（Josiah Conder，1852–1920）** 主持设计的洋馆 + 和馆 + 撞球室（ビリヤード室）三栋一体的本邸群完工，成为明治时代日本最具代表性的"近代邸宅"之一—— **洋楼主要用于接待客人，而岩崎家族的日常起居则在西边紧挨着的和馆之中**，洋楼与撞球室之间还有**地下通道相连（通道不公开）**。

**约西亚·康德尔（Josiah Conder）的来历**：康德尔 **1852 年出生于伦敦**，1877 年 25 岁来到日本任工部大学校（现东京大学工学部）教师。他来日之前曾在英国皇家建筑师协会举办的设计比赛中以"乡村别墅"方案获得第一名，被寄予厚望；但他归根到底只是刚进入设计事务所工作的年轻建筑师，是明治新政府大胆起用了这名年轻人。康德尔之所以愿意去日本，部分原因是他**曾在伦敦世博会上见识了日本绘画与民间工艺并被深深吸引**——"就像被日本动漫迷住的欧洲宅男一样"。康德尔作为教师培养出了**辰野金吾**（设计东京站丸之内站舍）、**片山东熊**（设计赤坂离宫迎宾馆 / 东博表庆馆）、**曾祢达藏**（设计庆应义塾大学图书馆）、**佐立七次郎**（设计日本邮船小樽支店）、**河合浩藏**（设计神户地裁）等近代建筑家。他也曾受明治政府之托设计**鹿鸣馆（1883）**，并拜画家**河锅晓斋**为师；任职期满后曾回英国但不久即返回日本，**娶日本女子为妻并在日本终老一生**。

**建筑风格判定**：旧岩崎邸的洋楼基调是 **17 世纪前期在英国风靡一时的雅各宾样式（Jacobean Revival）**—— 楼梯上每根柱子都缠绕着以**植物为主题的华丽装饰**，正是雅各宾样式的典型；阳台是**科洛尼亚式（Colonial Style）**；内部装饰则带有**文艺复兴 + 伊斯兰建筑特点**（每个房间天井阳台与暖炉上的瓷砖花纹各不相同，都是伊斯兰风格）。而位于建筑东侧的台球室"有人认为是**美国哥特风的木质建筑**，而设计师一开始模仿的其实是**瑞士的山中小屋**" —— 整座岩崎家旧宅可以说是"**古今东西各种风格的杂糅体**"，是 1896 年建成时的明治洋馆杰作。

**康德尔在日本近代建筑史的位置**：人称"**日本近代建筑之父**"也有人称他"**日本近代建筑之母**"。他在三菱、三井、古河等大财阀之间游刃有余地承接住宅 / 俱乐部 / 商社建筑订单（三菱 1 号馆 1894、三井俱乐部 1913、古河邸 1917 等），"**正是因为不拘泥于自己的风格，才更容易接纳对方的风格**"——这正是后世评价康德尔"近代建筑之母"的源由，也是"**康德尔风格的反讽**"：寻找所谓的"康德尔风格"会发现"每座都堪称一种风格的教科书，但距离共通的康德尔风格似乎越来越远"。

**岩崎邸的命运**：鼎盛期岩崎邸用地约 1.5 万坪、共 20 余栋建筑；战后用地被 GHQ 接收、之后归最高检察厅司法研修所与司法图书馆使用，建筑群仅存洋馆 + 和馆 + 撞球室 3 栋（合占原用地约 1/3，约 5,000 坪）。1961 年洋馆与撞球室被指定为 **重要文化财**，1969 年和馆大广间也被追加指定。1994 年所有权由司法部转交东京都，2001 年作为"旧岩崎邸庭园"对外开放。

## 看点

1. **洋馆（重要文化财，1896，约西亚·康德尔）**：木造 2 层 + 地下 1 层；进入砖墙围起来的区域后是一道斜坡，路尽头就是棕榈树掩映之下的两层木造洋楼；**小楼的正门在北边、右侧还有塔楼，整座建筑是左右不对称的结构**，从窗户四周环绕的华丽装饰即可看出主人非同一般。从正面玄关进入，**1F 和 2F 都以会客大厅为中心**，其他房间作为餐厅、集会室、客房和书房；每个房间的墙壁及天井装饰设计各不相同，均值得细看。
2. **楼梯上的植物主题缠柱装饰（雅各宾样式典型）**：洋楼设计者在楼梯上下了很大功夫，**每根柱子上都缠绕着以植物为主题的华丽装饰**——这种设计被称为**雅各宾样式（Jacobean Revival）**，曾于 17 世纪前期在英国风靡一时，是判定洋楼基调风格的核心物证。
3. **和馆大広间（重要文化财，1896，大河喜十郎）**：与洋馆通过短廊相连，纯日式 47 叠大広间 + 书院造；**和馆位于洋楼西边，是岩崎家族日常起居的空间**；与紧邻的洋馆形成"一墙之隔，两个文明"的并置奇观，是明治和洋折衷的标本。
4. **撞球室（ビリヤード室，重要文化财）**：单独的木造小屋，**美国哥特风的木质建筑**（设计师一开始模仿的其实是**瑞士的山中小屋**）+ Stick Style 木骨板墙；与洋馆通过**地下通道相连**（通道不公开）；内部当时罕见的"金唐革纸"（金箔压制压花壁纸）保存完好。
5. **装饰细节集锦（伊斯兰 + 文艺复兴 + 各种杂糅）**：
   - **屋檐内侧的镂空花纹** —— 即使是视线难以企及的屋檐内侧都有镂空花纹，体现设计者对装饰的执着；
   - **每个房间天井阳台与暖炉上的瓷砖花纹各不相同** —— 都是**伊斯兰风格**；
   - **蒸汽暖气片上的雕刻** —— 连蒸汽暖气片上都刻有装饰（"啊，天使！"），需贴近观察才能发现；
   - **科洛尼亚式阳台 + 印度伊斯兰风格圆顶塔楼**（康德尔在英国时期对东方建筑的兴趣）+ 内部彩色镶嵌玻璃 + 雕花石柱 + 英国制壁炉。
6. **庭园 + 大芝生（草坪）**：原 1.5 万坪江户大名屋敷池泉回游式庭园仅存约 1/3，但仍可见原有的池泉中岛 + 苔石 + 古树枝干；**面对草坪的阳台一侧**是南面拍照的最佳机位（详见下文）；草坪上 6 月有玉兰 / 紫阳花零星开放。
7. **约西亚·康德尔相关展示**：洋馆内有康德尔与岩崎家族关系的小型常设展、康德尔设计草图复刻 + 其他东京都内现存康德尔作品（三菱 1 号馆 1894 / 三井俱乐部 1913 / 古河邸 1917 / 鹿鸣馆 1883 已拆除）的展板介绍。

## 推荐玩法

从上野公园不忍口出发，沿不忍池西岸南行约 8 分钟可见旧岩崎邸正门；进门先在**入口的北边拍洋楼全景**（最佳拍摄地点 1），再从**南边拍面对草坪的阳台一侧**（最佳拍摄地点 2），**入馆需脱鞋**（套鞋袋会派发）—— 顺时针动线：洋馆 1F 客厅 → 食堂 → 通过短廊进和馆大広间 → 折返出和馆 → 庭园草坪绕一周 → 撞球室 → 洋馆 2F 寝室 → 出口；**Day 3 安排压缩为 45 分钟**（脱鞋 + 三栋楼 + 庭园基本步骤；如时间充裕可留 ≥ 1 小时细看 2F 寝室与所有伊斯兰花纹瓷砖）。庭园内禁止携食物入园；茶屋"喫茶 棟梁"在洋馆出口侧售抹茶 + 和菓子套餐 ¥800。

## 建筑专属字段

- **建筑师 + 建成年份**：
  - 洋馆 + 撞球室：1896 年（明治 29 年），**Josiah Conder（乔赛亚・康德）** 设计 —— 英国人，1877 年受聘来日担任工部大学校教授，被誉为"日本近代建筑之父"，门下培养辰野金吾（东京站丸之内站舍）、片山东熊（迎宾馆赤坂离宫 / 东博表庆馆）、曾祢达蔵等明治第一代日本建筑师；旧岩崎邸为其私邸作品代表。
  - 和馆大広间：1896 年同期由日本建筑师 **大河喜十郎** 设计（书院造 + 大広间）。
- **设计要点**：
  - 洋馆采**詹姆斯一世样式**（Jacobean Revival）+ 木造桁架阳台 + 印度伊斯兰风格圆顶塔楼（Conder 早期对东方建筑的迷恋）+ 内部彩色镶嵌玻璃 + 雕花石柱 + 英国制壁炉。
  - 撞球室采**瑞士山小屋样式**（Swiss Chalet Style）+ 美国 Stick Style 木骨板墙 + 内部"金唐革纸"金箔压花壁纸；通过地下通道连通洋馆。
  - 和馆与洋馆通过短廊相连，纯日式书院造，与洋馆形成"和洋一墙之隔"的明治折衷标本。
- **可否进入参观**：洋馆 + 和馆 + 撞球室 3 栋**均开放游客脱鞋入内**；庭园开放散步。**洋馆 2F 寝室部分房间因保护需要不开放**。
