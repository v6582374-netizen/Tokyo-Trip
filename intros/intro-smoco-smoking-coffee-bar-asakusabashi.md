# SMOCO 吸烟咖啡店 浅草桥店（SMOCO SMOKING&COFFEE BAR 浅草橋店 / SMOCO Smoking & Coffee Bar Asakusabashi）

- **类型**：餐饮（吸烟咖啡店）
- **位置**：东京都台东区浅草桥（浅草桥站附近）；最近车站 JR 总武线 / 都营浅草线「浅草桥」站步行约 2 分钟。
- **历史与背景**：SMOCO SMOKING&COFFEE BAR 浅草橋店是一家允许吸烟的咖啡店，位于浅草桥站附近。在日本 2020 年《改正健康増進法》全面实施后，室内全面禁烟成为主流，SMOCO 作为少数合法设置吸烟区 + 提供精品咖啡的复合型店铺，为吸烟者提供了一个可以同时享受咖啡与香烟的合规空间。
- **看点**：
  1. **合规吸烟 + 精品咖啡复合空间**：在东京都内全面禁烟的大环境下，SMOCO 提供合法的室内吸烟环境 + 手冲 / 意式咖啡。
  2. **浅草桥站步行圈内**：与 Day 7 动线（两国 → 浅草 → 押上）顺路，可作为退房后 / 浅草段前的短暂休憩点。
- **票务 / 营业时间**：
  - 营业时间：待用户实地确认（一般咖啡店营业时间约 08:00–20:00）。
  - 定休日：待用户实地确认。
- **交通到达**：
  - **从两国 COGO RYOGOKU 出发**：都营浅草线 两国 → 浅草桥 1 站 / 约 2 分钟 / ¥180（TST ❌）→ 步行约 2 分钟。合计 ≈ 6 分钟。
  - **从浅草雷门出发**：都营浅草线 浅草 → 浅草桥 2 站 / 约 4 分钟 / ¥180（TST ❌）→ 步行约 2 分钟。
- **周边联动**：步行 5 分钟内可达 都营浅草线浅草桥站（两国方向 / 浅草方向均 1–2 站）；与 Day 7 浅草段（雷门 / 仲见世通）及两国段（江户东京博物馆 / 江户荞麦 细川）通过都营浅草线一线串联。
- **消费档次**：人均 **¥500–1,000（JPY）**（咖啡 + 轻食）。
- **注意事项**：
  - 店内允许吸烟，非吸烟者可能不适应烟味环境。
  - 具体营业时间 / 定休日以现场或 Google Maps 实时信息为准。
