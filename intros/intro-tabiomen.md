# Tabio MEN（Tabio MEN / TabioMEN）


- **类型**：购物（袜子专门店，男士线；4 个补丁外观点之一，按地理纳入 Day 6 上午段）
- **位置**：东京都涩谷区 / 港区表参道沿线 Tabio 男士线分店；最近车站 JR 山手线 / 东京 Metro 半蔵门线 / 副都心线 / 银座线「涩谷」站 + 千代田线 / 银座线「表参道」站均可达（TST ✅ Metro 段、TST ❌ JR 段）。地址按当日实际访问店铺为准 —— 待用户实地确认（官网：https://tabio.com/jp/）。
- **历史与背景**：Tabio（タビオ）是 1968 年由越智直正（Naomasa Ochi）在大阪创立的"袜子专门店"品牌，原名「ダン株式会社」（Dan Co., Ltd.），2006 年改商号为现在的「Tabio Corporation」。Tabio 是日本"袜子消费"现代化的代表企业之一——上世纪 70–80 年代日本袜子行业由百货公司散卖、价格无差异化主导，越智直正在大阪开设第一家"袜子单一品类专门店"模式（每家店 200–300 SKU、按用途 / 材质 / 颜色细分），把袜子从"附属品"转化为"独立商品"。Tabio 在 2000 年代起进入东京 / 横滨 / 名古屋的核心商圈（涩谷 / 表参道 / 新宿 / 银座 / 横滨港未来 / 名古屋 PARCO 等），并按「Tabio（中性 / 女士主线）」「Tabio MEN（男士线）」「Tabio CHIBI（童装线）」「Tabio Sports（运动线）」分线。**Tabio MEN** 是男士袜专门店，主打商务袜（黑色 / 深蓝 / 灰色三色 + 5 种线密度）+ 商务休闲袜（图案 / 棉麻 / 羊毛 / 莱卡）+ 个性款（视觉系图案 / 漫画联名 / 运动 logo），价格区间 ¥800–3,000 / 双；与全棉 / 棉混纺为主的全球快消袜（Uniqlo / GU 等 ¥390–590 / 双）相比，Tabio MEN 的纺织密度 + 缝合工艺明显更耐穿（用户口碑 6 个月以上），是涩谷 / 表参道一线"日本制造袜子"代表品牌。
- **看点**：
  1. **男士商务袜 5 种线密度**：黑 / 深蓝 / 灰色三色 × 5 种线密度（轻薄夏季 / 普通四季 / 厚款冬季 / 商务双层 / 商务长筒）。
  2. **联名款 + 视觉系图案**：与漫画 / 偶像 / V-kei 视觉系联名的限定款图案，¥1,500–3,000 / 双。
  3. **Tabio MEN「DRESS UP」系列**：高端商务款（西阵织 / 高密度棉绢 / 阴丹士林染 / 限定海岛棉），¥2,500–5,000 / 双。
  4. **店内的"袜架尺寸标识"**：每款袜子按男士尺码 25–28 cm 分四档，店员可按客户脚长推荐。
- **推荐玩法**：16:50 从 RAGTAG / Tabio 一线沿表参道方向逛过来 → Tabio MEN 速逛 10–15 分钟（重点：商务袜 + 一双联名款图案）；如时间紧张可与 Tabio 一并外观打卡跳过。
- **票务 / 营业时间**：
  - 入场料金：免费进入；商品消费另计。
  - 营业时间：11:00–20:00（部分店延长至 21:00）；定休日 不定休。
  - 6/8（周一）正常营业。票价 / 营业时段如有调整以官网为准 —— 待用户实地确认（官网：https://tabio.com/jp/）。
- **交通到达**：
  - 从前一地点（按外观打卡顺序，从 RAGTAG / Tabio 一线步行）：步行 5–10 分钟（视当日访问店铺位置而定），TST 不适用（步行）。
  - 从两国 COGO RYOGOKU 出发：都营大江户线 → 半蔵门线 涩谷 / 表参道（约 25 分钟、TST ✅）+ 步行 5–10 分钟，0 元票价（TST 已覆盖）。
- **周边联动**：步行 1 分钟可达 **Tabio**（同品牌中性 / 女士主线）；步行 5 分钟内可达 **RAGTAG**（涩谷店设计师品牌中古）；步行 8 分钟可达 **Euro Live**（道玄坂中段，已结束仅外观）。
- **消费档次**：商务袜 ¥800–3,000 / 双；联名 / 限定款 ¥1,500–3,000 / 双；DRESS UP 系列 ¥2,500–5,000 / 双；本次预算约 ¥0–4,000（按机动选购 1–2 双）。

  - 现金 / 主要电子支付（Suica / PASMO / 主要信用卡）皆可。

  - **价格区间**：¥800–5,000 / 双（商务款 ¥800–3,000、限定 / DRESS UP ¥2,500–5,000）。
